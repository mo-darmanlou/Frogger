# Frogger

A simple clone of the classic 1981 Konami arcade game, "Frogger," developed as the final project for the Fundamentals of Programming course.

<img width="600" alt="usage_example" src="https://techland.time.com/wp-content/uploads/sites/15/2012/05/frogger.jpg?w=720&h=480&crop=1" />
## Project Structure

*   `main.cpp`: The main source code for the game.
*   `Audios/`: Contains sound effects used in the game.
*   `Fonts/`: Contains fonts for displaying text like scores or titles.
*   `Images/`: Contains all the visual assets, including vehicles, logs, and background textures.
